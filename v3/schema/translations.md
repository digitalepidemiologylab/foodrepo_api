# Translations Object

Translations, where not all supported languages are present.

### Schema

* `de`
  * Type: `string`
  * Description: The German translation.
* `fr`
  * Type: `string`
  * Description: The French translation.
* `it`
  * Type: `string`
  * Description: The Italian translation.
* `en`
  * Type: `string`
  * Description: The English translation.
